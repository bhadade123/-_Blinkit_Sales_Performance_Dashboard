# -_Blinkit_Sales_Performance_Dashboard

## 📊 Overview
The **Blinkit Dashboard** is a Power BI project that visualizes key business metrics for Blinkit — *India’s Last Minute App*.  
It provides insights into **sales performance**, **item categories**, **outlet distribution**, and **customer ratings** through interactive charts and filters.

---

## 🚀 Features
- **Total Sales**: Displays overall revenue (1.20M).  
- **Average Sales**: Shows mean sales per outlet (141).  
- **Number of Items**: Tracks total product count (9K).  
- **Average Rating**: Reflects customer satisfaction (4/5).  
- **Outlet Establishment Trend**: Line chart showing growth from 2010–2020.  
- **Fat Content Analysis**: Compares low-fat vs regular product sales.  
- **Item Type Breakdown**: Highlights top-selling categories like fruits, snacks, and household goods.  
- **Outlet Size & Location**: Visualizes performance by outlet tier and size.  
- **Visibility Table**: Summarizes sales, ratings, and item visibility by outlet type.

---

| Component | Description |

---

## 📈 Insights
- Tier 3 outlets lead with **472K total sales**.  
- Regular fat items outperform low-fat ones.  
- Fruits & Vegetables and Snack Foods are top-selling categories.  
- Supermarket Type1 dominates with **788K total sales** and **338.65 visibility score**.
